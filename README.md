# babel-plugin-log

For the lazy people who's tired of writing out 'console' for every log they want to do...

> Just a plugin for learning purposes. Not published on NPM (yet).


## Example

**In**

```js
log('hello world');
```

**Out**

```js
console.log('hello world');
```

## Usage

### Via `.babelrc` (Recommended)

```json
{
  "plugins": ["log"]
}
```
